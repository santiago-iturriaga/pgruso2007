Condiciones para el usuario linux de un cliente


  • El usuario debe estar creado en el sistema. Puede utilizar el comando adduser "usuario" para crear un usuario
  • El usuario debe tener permisos de escritura en la carpeta {RAIZ} o Perteneser al grupo {GRUPOFENTON}
  • El usuario de Apache debe poder conectarse por ssh con el usuario. Crear en el home del usuario la carpeta .ssh y agrear la clave publica al archivo ".ssh/authorized_keys2"
  • Dependiendo de la configuracion del cluster el usuario debe ser creado en todos los nodos.
  • Si creo un nuevo usuario recuerde asignarle un password
